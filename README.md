Simply change to the frontend or backend directly and run `npm install`.

Install the required dependencies:
Frontend:

Documentation:
This project uses VitePress to host its documentation, install it with:

Then run it with:
`npm run docs:dev`

- bcryptjs

- express
- jsonwebtoken
- mongodb
- mongoose
- axios
- cors
- google-auth-library
- multer
